# Análise de Dados de Acidentes Rodoviários

## Objetivo
Este projeto tem como objetivo analisar dados de acidentes rodoviários para identificar padrões relacionados à gravidade, clima, horário e principais causas.

## Problema
Compreender quais fatores estão mais associados à ocorrência e gravidade de acidentes rodoviários e gerar insights para prevenção.

## Ferramentas utilizadas
- Python
- Pandas
- Análise Exploratória de Dados
- Visualização de Dados

## Etapas do projeto
1. Coleta e organização dos dados
2. Análise exploratória
3. Identificação de padrões
4. Visualização gráfica dos resultados
5. Geração de recomendações

## Resultados
- Identificação dos tipos de acidentes mais frequentes
- Relação entre condições climáticas e gravidade
- Horários com maior incidência de acidentes
- Recomendações baseadas em dados para prevenção

## Arquivos do projeto
- Relatório completo da análise
- Gráficos gerados a partir dos dados
- Documentação do estudo
